# Superstore Sales Analysis & EDA
## Objective
- Analyze sales, profit, and discount impact on different products, regions, and customer segments.
- Identify top products and regions.
- Prepare the dataset for potential Machine Learning models (e.g., predicting sales or profit).
## Tools
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- ## Dataset
- [Kaggle Superstore Dataset](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)
## Highlights
- Top selling and profitable products
- Regional sales and profit analysis
- Discount impact on sales and profit
- Monthly and yearly sales/profit trends
